# Colors Wheels

## Motivation

Inspired by [Felipe Pantone's](https://www.felipepantone.com/) kinetic color wheel [installations](https://www.instagram.com/reel/CzV_9kcoRdO/), I wanted to mimic a similar playful, colorful experience in the browser.

## Interactive Browser Color Wheels

import InteractiveColorWheel from "../../../src/components/InteractiveColorWheel";

### Concept

Let's start with a basic HSL color _wheel_ over an HSL color _square_.

Move the rotation slider to rotate the wheel from 0 to 360 degrees.
